How windows work
⚡ P1–P2 tasks are placed as early as possible within the window — don't wait for idle time.
🔍 P3 tasks hunt for idle gaps in the first half of the window, then earliest available after.
💤 P4 tasks fill any idle gap anywhere in the window.
⚠ If no slot is found before the window closes, the task is flagged on the schedule for manual reassignment.
